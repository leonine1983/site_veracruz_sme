Elementos HTML Inline

HTML define uma longa lista de tags inline, uma lista completa pode ser encontrada na Mozilla Developer Network.

  • Para deixar o texto em negrito, use <strong>.
  • Para deixar o texto em itálico, use <em>.
  • Abreviações, como HTML, devem usar <abbr>, com um atributo title opcional para a frase completa.
  • Citações, como — Mark Otto, devem usar <cite>.
  • Texto apagado deve usar <del> e texto inserido deve usar <ins>.
  • Texto em sobrescrito exemplo usa <sup> e texto em subscrito exemplo usa <sub>.

A maioria desses elementos são estilizados pelos navegadores com poucas modificações da nossa parte.
