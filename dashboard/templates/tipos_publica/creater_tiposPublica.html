Para adicionar uma tag <i> como ícone, siga estes passos:

  1. Clique no botão abaixo.
  2. Acesse o site FontAwesome.
  3. No campo de busca, digite o nome do ícone desejado em inglês (por exemplo: house, pencil, category).
  4. Clique no ícone correspondente e, em seguida, copie a tag semelhante a esta:
        <i class="fa-solid fa-house"></i>

Retorne ao dashboard e cole a tag copiada no campo "Adicionar TAG <i> do FontAwesome".
